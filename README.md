# Gentle Phone Transfer — Windows EXE builder (Premium SaaS UI)

This project builds a Windows app for **Gentle Phone Transfer** using Tauri.

## Build without coding (GitHub Actions)
1) Create a GitHub repo (private is fine).
2) Upload all files from this zip into the repo (drag & drop on GitHub).
3) Go to **Actions** → **Build Windows EXE** → **Run workflow**.
4) When finished, download the artifact **GentlePhoneTransfer-Windows**.
5) Inside the artifact, find the installer/exe produced by Tauri and rename it to:
   **Gentle Phone Transfer.exe** (optional)

## Use
- Open the app
- Site: https://artropical.com
- Pairing code: from WP Admin → GentleSite → Phone Transfer

This MVP supports pairing + heartbeat. Transfers + iPhone scan come next.
